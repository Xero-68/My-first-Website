## üé® 1. Change Background Color

```css
body {
  background-color: lightblue;
}
```

### How to use in HTML:

```html
<!DOCTYPE html>
<html>
<head>
  <style>
    body {
      background-color: lightblue;
    }
  </style>
</head>
<body>
  <h1>Hello, world!</h1>
</body>
</html>
```

## üé® 7. Rainbow Text with Classes

```css
.red { color: red; }
.orange { color: orange; }
.yellow { color: yellow; }
.green { color: green; }
.blue { color: blue; }
.purple { color: purple; }
```

### How to use in HTML:

```html
<!DOCTYPE html>
<html>
<head>
  <style>
    .red { color: red; }
    .orange { color: orange; }
    .yellow { color: yellow; }
    .green { color: green; }
    .blue { color: blue; }
    .purple { color: purple; }
  </style>
</head>
<body>
  <h1>
    <span class="red">R</span>
    <span class="orange">A</span>
    <span class="yellow">I</span>
    <span class="green">N</span>
    <span class="blue">B</span>
    <span class="purple">O</span>
    <span class="red">W</span>
  </h1>
</body>
</html>
```
